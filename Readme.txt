Meet The Family:

Problem:
    Given a ‘name’ and a ‘relationship’, you should output the people corresponding to the relationship in the order in
    which they were added to the family tree. Assume the names of the family members are unique.

    You should be able to add a child to any family in the tree through the mother. 

Relationship handled:

    Paternal-Uncle
    Maternal-Uncle
    Paternal-Aunt
    Maternal-Aunt
    Sister-In-Law
    Brother-In-Law
    Son
    Daughter
    Siblings

Running the problem with the below steps:

    1. use commands 'npm install' and 'npm start --silent <path of input.txt file>'
    2. Output for the test cases provided in input text file is printed in console.

Running the tests:
    1. use 'npm test' to see the execution test cases.
    2. To see the coverage goto coverage->lcov-report->index.html and open the file in browser.
        you can see the lines covered by test cases and % of coverage.
